# CAN-Bus-Analyzer-Dashboard
A CAN Bus Analyzer Dashboard is a monitoring and diagnostic interface used in automotive and industrial systems to visualize and analyze data communication over the Controller Area Network (CAN). It captures real-time signals from ECUs 

The dashboard helps users monitor CAN IDs, data frames, message frequency, timestamps, and live bus traffic in real time. It is designed for automotive diagnostics, ECU communication testing, embedded system development, and CAN protocol learning. The project supports Linux/Ubuntu environments and can be integrated with virtual CAN (vcan) or physical CAN hardware interfaces.

# Features
Real-time CAN message monitoring
Live dashboard visualization
CAN ID and data frame analysis
Timestamp and message frequency tracking
SocketCAN integration in Ubuntu/Linux
User-friendly graphical interface
Support for virtual CAN (vcan0) and physical CAN devices
Automotive ECU communication testing
